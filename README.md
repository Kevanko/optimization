# optimization

Coursework and lab assignments on fault-tolerant computing systems, MPI experiments, scheduling, game-theoretic models, and reliability analysis.

## Overview

This repository contains practical assignments related to parallel and distributed computing, performance evaluation, scheduling, and system reliability.

## Structure

- `lab1` - MPI communication and throughput experiments
- `lab2` - scheduling and strip packing tasks
- `lab3` - game-theoretic modeling
- `lab4` - reliability analysis with structural redundancy
- `lab5` - survivability of distributed computing systems
- `docs` - assignment statements and supporting materials

## Stack

Python, C/C++, MPI, Makefile, plotting utilities.

## Notes

This repository was created for academic coursework and lab submissions.
